AI Image Generator

A modern, responsive web application built with **React** and **Vite** that generates images using AI. This project uses **Tailwind CSS** for styling and is optimized for a fast development experience.


Features

* Prompt-to-Image: Generate unique images from text descriptions.
* Fast Performance: Powered by Vite for instant Hot Module Replacement (HMR).
* Responsive Design: Optimized for mobile, tablet, and desktop views.
* Modern UI: Styled with Tailwind CSS and PostCSS.

Tech Stack

* Frontend: [React.js](https://reactjs.org/)
* Build Tool: [Vite](https://vitejs.dev/)
* Styling: [Tailwind CSS](https://tailwindcss.com/)
* Language: JavaScript & TypeScript
